# dotfiles

my dotfiles for mac-os.

## Installation

  1. Install Command Line Tools:

     ```console
     xcode-select --install
     ```

  2. Clone the repository:

     ```console
     git clone --recursive git@github.com:uqichi/dotfiles.git
     ```

  3. Configure it:

     ```
     make install
     ```

## Command Usage

```
make
```
